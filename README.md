# 🎢 Simulador de Montaña Rusa – Loop Vertical

## 📌 Descripción

Este proyecto consiste en un simulador interactivo de una montaña rusa con un loop (rizo vertical), desarrollado como apoyo a un proyecto de física.

El objetivo es analizar si un carrito puede completar el loop dependiendo de la altura inicial desde la que se suelta.

---

## 🧠 Fundamento físico

Para que el carrito complete el loop, debe cumplirse la siguiente condición:

* La altura inicial debe ser al menos 2.5 veces el radio del loop.

Esto se basa en principios de:

* Conservación de la energía
* Movimiento circular

---

## 📊 Datos del proyecto

* Radio del loop: **0.43 m**
* Altura utilizada: **1.66 m**
* Altura mínima requerida: **1.08 m**

✔ El diseño cumple con la condición necesaria para completar el loop.

---

## 🎮 Funcionalidades

### 🔹 Pantalla 1: Caso correcto

* Muestra la simulación con los valores reales del proyecto.
* El carrito completa el loop.
* Se visualizan los datos físicos.

### 🔹 Pantalla 2: Interacción

* El usuario puede modificar la altura inicial.
* El sistema evalúa si el carrito completa el loop.
* Se muestra un mensaje indicando si funciona o no.
